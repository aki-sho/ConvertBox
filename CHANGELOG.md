# Changelog

ConvertBoxの主な変更をバージョンごとに記録します。

## 1.0.0 - 2026-07-19

最初の正式リリース。旧アプリ名からConvertBoxへ名称を統一。

- Video変換: MP4 / MOV / WebM / AVI
- Audio変換: MP3 / WAV / M4A / AAC
- Image変換: JPG / PNG / JPEG / GIF / WebP / SVG
- 変換前のファイル解析とメタデータ表示
- 変換と保存先選択を分離した操作フロー
- Windows向けインストール不要ポータブルEXE
- 設定、データ、ログ、キャッシュ、一時ファイルのポータブル保存
- 終了時のFFmpeg子プロセスと一時ファイルのクリーンアップ
- 二重起動の防止
- アプリ内バージョン表示
- 専用フォルダとREADMEを含むポータブルZIP生成
- EXEとZIPのSHA-256チェックサム生成
